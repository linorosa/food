1. Whisk together

```
3 tbsp apple cider vinegar
2 tbsp sugar
1 cup mayonnaise
S&P
```

2. In a large bowl, combine

```
red onion, diced
dried fruit
sunflower seeds
bacon bits
```

3. Thaw some frozen broccoli a few hours before, and mix before eating.

```
frozen broccoli
```
